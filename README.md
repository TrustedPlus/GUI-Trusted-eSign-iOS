# Пример приложения на React Native ##

----------
## Сборка:
Установить node и зависимостей
```
brew install node
npm install -g react-native-cli
npm install

Удалить папку node_modules/@types/node
```


## Запуск приложения:

### IOS:
```
npm run start:ios

```

### Android:
```
npm run start:android

```
